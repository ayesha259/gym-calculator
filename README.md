# BMI Calculator

This repository contains two versions of a BMI calculator:

- `bmi-calcualter.py` — original Tkinter GUI
- `bmi_streamlit.py` — Streamlit web app version

Run the Streamlit app:

```powershell
& "C:/Users/Dell/Downloads/New folder/.venv/Scripts/python.exe" -m pip install -r "c:/Users/Dell/Downloads/New folder/requirements.txt"
& "C:/Users/Dell/Downloads/New folder/.venv/Scripts/python.exe" -m streamlit run "c:/Users/Dell/Downloads/New folder/bmi_streamlit.py"
```
